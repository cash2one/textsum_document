data.ExampleGen(data_path, num_epochs=None)¶

Generates tf.Examples from path of data files.

Binary data format: <length><blob>. <length> represents the byte size of <blob>. <blob> is serialized tf.Example proto. The tf.Example contains the tokenized article text and summary.
Args:
data_path: path to tf.Example data files. num_epochs: Number of times to go through the data. None means infinite.
Yields:
Deserialized tf.Example.

If there are multiple files specified, they accessed in a random order.
